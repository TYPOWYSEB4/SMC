Mod Pack Description

Fabric+ is a mod pack for Vanilla servers to improve and optimize gameplay
Pack is based on Fabric Loader for the latest official version 1.19.2
Mod list is located in section below installation

Pack Installation

  • To install mods, Mod Loader is required
  • We need Fabric, you download it from given link. On site choose "Download installer (Windows/.EXE)".
  • After downloading file, click run then choose Minecraft version 1.19.2 and Loader version 0.14.9 When you did this click Install.
  • After sucessful Loader installation we can add our mods. Download Pack the section below and put it in to .minecraft/mods
  • To finish extracting compressed file using WinRAR click RMB on Pack, then click "extract here"
  • After opening Launcher you need to choose profile with name fabric-loader-1.19.2
